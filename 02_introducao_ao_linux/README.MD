# FUNDAMENTOS DE LINUX

- Criado em 1991 por linus Torvals

- Linux é o Kernel ou nucleo do sistema operacional. Ou seja ele não é o sistema operacional mas sim o que da vida ao sistema operacional

- O linux é multitarefa e multiusuário, ou seja ele consegue processar uma ou mais processamentos ou usuários ao mesmo tempo.

- É desenvolvido por diversas pessoas e empresas ao redor do mundo.
